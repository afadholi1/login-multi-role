# Multi-Role Auth System (RBAC)

Project ini adalah aplikasi Fullstack JavaScript untuk mempelajari sistem autentikasi modern yang aman.

### Fitur Utama:
* **Multi-Role:** Admin dan User (Role-Based Access Control).
* **Security:** JWT Access Token & Refresh Token (HttpOnly Cookie).
* **Database:** MySQL dengan Sequelize ORM.
* **Backend:** Node.js & Express.
* **Frontend:** React JS (Coming Soon).

### Cara Menjalankan:
1. Clone repo ini.
2. `cd backend` lalu `npm install`.
3. Buat database `db_multi_role` di MySQL.
4. Sesuaikan file `.env`.
5. Jalankan `nodemon index.js`.
